# Buchnancials

A privacy-first, local cash flow tracking tool based on bank CSV exports.

Buchnancials helps you understand where your money comes from and where it goes — with minimal setup, full control over your data, and clear visualizations.

## Features

- Import bank transactions from CSV
- Flexible column mapping via preview before import
- Monthly transaction overview
- Custom categories for income and expenses
- Manual and editable categorization
- Exclude transactions (e.g. internal transfers, investments)
- Rule-based automatic categorization (heuristics)
- Sankey diagrams for monthly cash flow visualization
- Aggregated views (monthly, quarterly, yearly)
- Local-first: all data stored on your machine

## Principles

- Privacy-first (no external APIs, no cloud)
- Simple and fast workflow
- Transparent and editable data

## Status

Early-stage side project / MVP.
